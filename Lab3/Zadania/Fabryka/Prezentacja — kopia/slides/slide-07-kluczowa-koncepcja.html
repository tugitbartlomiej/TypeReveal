Kluczowa Koncepcja: pokazSpecjalneFunkcje()

Przed (bez interfejsu):

BMW bmw = new BMW();
bmw.aktywujTrybM();  // ← BMW ma tę metodę

Koenigsegg k = new Koenigsegg();
k.aktywujTryb1600KM();      // ← Koenigsegg ma 3 metody
k.wlaczSystemNitro();        // ← Wszystkie różne!
k.sprawdzPredkoscMaksymalna(); // ← Jak uogólnić?
↓

Po (z interfejsem):

Samochod bmw = new BMW();
Samochod k = new Koenigsegg();

// JEDNA metoda dla wszystkich!
for (Samochod auto : salon) {
    auto.pokazSpecjalneFunkcje();
    // ↑ BMW wywołuje 1 funkcję
    // ↑ Koenigsegg wywołuje 3 funkcje
    // → To jest POLIMORFIZM!
}

To jest POLIMORFIZM!

Jedna metoda w interfejsie → różne implementacje w klasach
